// PulseBar

PulseBar is a macOS utility that surfaces process activity in a clean, glanceable UI. It lets you filter, search, sort, and act on running processes with a streamlined SwiftUI experience.

## Features
- Filter tabs for common process groups
- Fast search across name, bundle, path, and ports
- Sortable columns (Name, CPU, Memory, Uptime, Ports)
- Column visibility picker
- Multi-select mode with bulk force-quit
- Modern SwiftUI design with keyboard- and mouse-friendly interactions

## Requirements
- macOS 13+
- Xcode 15+

## Building
1. Open the Xcode project/workspace in Xcode.
2. Select the PulseBar scheme.
3. Build and run.
